# React Native Hello App

A simple React Native application with a hello page.

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- React Native CLI
- For iOS: Xcode
- For Android: Android Studio

### Installation

1. Install dependencies:
```bash
npm install
```

### Running the App

#### For iOS:
```bash
npm run ios
```

#### For Android:
```bash
npm run android
```

#### Start Metro Bundler:
```bash
npm start
```
